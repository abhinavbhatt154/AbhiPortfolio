My Projects

BlinkBasket

BlinkBasket is a conceptual e-commerce platform that reimagines how users shop when the same product is available from multiple sellers with different delivery speeds and prices. Instead of showing only one seller, BlinkBasket allows users to compare:

Quick Delivery – delivered in about 1 hour from a nearby local shop.

Same Day Delivery – delivered later the same day from a seller within the city.

All India Delivery – delivered in 2–3 days from another city at a lower price.

HTML5 CSS3 JavaScript (ES6+) Bootstrap 5

PhishGuard

PhishGuard is an Android cybersecurity application that detects potentially fraudulent SMS messages in real time using keyword-based phishing analysis. It alerts users through instant notifications and maintains a local history of detected threats using SQLite. Built with Java and Android Studio, the project demonstrates Android Broadcast Receivers, runtime permissions, notifications, and local database integration while focusing on improving mobile security awareness.

Java Android Studio
